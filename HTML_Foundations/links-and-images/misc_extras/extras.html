MORE HTML ATTRIBUTES

Now that we're more comfortable with HTML attribute syntax, we can add some finishing touches to our HTML boilerplate code. Every web page you create should define the language it's written in and it's character set.

DOCUMENT LANGUAGE

A web page's default language is defined by the lang attribute on the top level html element. Out document is in English, so we'll use the en country code as the attribute value. If you aren't sure on your country code, look it up here country codes

CHARACTER SETS

A character set is kind of like a digital alphabet for your browser. It's different from the language your document in that it only affects how the letters themselves are rendered, not the language of the content.

  1. bir
  2. iki
  3. üç
  4. dört
  5. beş

When this is viewed in your browser, you can sometimes see weird stuff with the characters. This is because the default character set for most browsers don't accommodate these "special" characters. To fix this you specify UTF-8 character encoding by adding a meta element with a charset attribute to the head of the file

These days, UTF-8 is a sort of universal alphabet for the Internet. Every web page you create should have this line in its head.
